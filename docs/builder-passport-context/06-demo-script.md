# Demo Script

## Goal
Show that hackathon participation, contribution, and wins can become a reusable public credential.

## Demo flow
1. Open landing page
2. Explain the problem
3. Connect builder wallet
4. Create Patrick Passos profile
5. Open Monad Blitz event page
6. Join the event
7. Use peer wallet to attest Patrick
8. Use organizer wallet to assign an official award
9. Open Patrick's final passport page
10. Show AI summary
11. Invite audience to add live attestations if time allows

## 60–90 second talk track
Today, when someone participates in or wins a hackathon, that usually becomes a post on LinkedIn and then disappears. Real contribution is fragmented and hard to verify.

Builder Passport turns hackathons into portable public credentials. A builder creates a profile, joins an event, receives peer attestations like Technical or Helpful, and can receive official awards from the organizer like Winner or Best Technical Solution.

Now the hackathon does not end as a social post. It becomes a reusable reputation layer.

## Important presentation rule
Do not depend on the audience for the core flow.
Use prepared wallets first. Audience interaction is a bonus only.
