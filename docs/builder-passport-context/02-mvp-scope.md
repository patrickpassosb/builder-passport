# MVP Scope

## In scope
- Connect wallet
- Create builder profile
- Create or load one hackathon event
- Join the hackathon event
- Peer attestation to another participant
- Organizer award assignment
- Public passport page
- AI-generated summary based on counts and awards

## Out of scope
- Feed or social timeline
- Comments system
- Chat
- Multi-event search
- Global rankings across many hackathons
- NFT / token / payments
- OAuth login
- GitHub API integrations
- LinkedIn API integrations
- Uploads / media storage
- Complex admin system

## Strict scope rule
Build for **one featured event** in the demo:
- Monad Blitz São Paulo 2026

Do not try to build a full platform for all hackathons.

## Success criteria
The demo is successful if the app can show this flow:
1. Create Patrick Passos profile
2. Join Monad Blitz event
3. A second wallet gives a peer attestation
4. An organizer wallet assigns an official award
5. The passport page shows the final reputation state
