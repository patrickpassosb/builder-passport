# PROJECT_CONTEXT

Project: Builder Passport

Thesis:
Builder Passport turns hackathon participation, peer-recognized contribution, and official results into a portable public credential.

MVP:
- create profile
- join one hackathon
- peer attestation
- organizer award
- public passport page
- AI summary

Three pages only:
- landing
- hackathon page
- passport page

Core contract features:
- profiles
- hackathons
- join status
- attestation counts
- awards

Design principle:
Do not build a full hackathon platform.
Build the reputation layer that a future hackathon platform would need.

Demo target:
Patrick Passos creates a passport, joins Monad Blitz, receives a peer attestation, receives an official award, and shows the final passport page.
