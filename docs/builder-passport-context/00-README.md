# Builder Passport — Context Pack

This folder is a handoff pack for a CLI coding agent.

Goal: build a hackathon MVP called **Builder Passport** for Monad Testnet.

Core product:
- Create a builder profile
- Join one hackathon
- Receive peer attestations
- Receive official organizer awards
- Show a public passport page
- Generate a short AI summary from the onchain data

Do not expand scope beyond the MVP.

Recommended reading order for the agent:
1. `09-hackathon-constraints.md`
2. `01-project-idea.md`
3. `02-mvp-scope.md`
4. `04-smart-contract-spec.md`
5. `05-ui-spec.md`
6. `07-build-order.md`
7. `08-prompt-pack.md`

Suggested stack:
- Next.js
- TypeScript
- Tailwind CSS
- viem / wagmi
- Solidity
- Monad Testnet

Preferred product positioning:
> Builder Passport is a portable proof of participation, contribution, and wins in hackathons.
